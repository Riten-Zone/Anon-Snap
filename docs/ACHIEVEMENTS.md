# Achievements

What's been built so far in Anon Snap.

## Capture & Privacy
- 100% offline — no data leaves the device
- On-device face detection (MLKit), no internet required
- Front and back camera support
- Import photos from the device gallery

## Stickers
- 21 built-in stickers across 5 themed collections
- Automatic face detection with Hypurr stickers applied by default
- Customizable default sticker for auto-detected faces
- Custom sticker upload with automatic background removal (iOS VisionKit)
- Snapchat-like sticker editing: move, rotate, scale, delete
- Magnifier loupe while dragging stickers
- Switch stickers — replace individual or all stickers at once
- Add stickers manually to any part of the image
- Randomize by collection or all stickers at once
- Full feature parity between built-in and custom stickers, including randomize

## Editing
- Freehand drawing mode
- Undo/redo for all editing actions

## Export & Sharing
- Save to device Camera Roll (dedicated "Anon Snap" album)
- Native iOS share sheet
- Direct share to Twitter and Telegram

## App Infra & Release
- Splash/launch screen
- iOS 17.0 minimum target, React Native 0.83 (New Architecture)
